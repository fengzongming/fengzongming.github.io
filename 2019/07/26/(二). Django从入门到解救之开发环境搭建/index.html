开发环境搭建

俗话说：工欲善其事必先利其器。所以在学习Django项目之前，需确保我们的开发环境已搭建完成。

  1. 确保已经安装Python 3.6以上的版本，本次学习以Python 3.6版本进行练习。

  2. 安装virtualenvwrapper，这个是用来创建虚拟环境的包(请参考上一篇博文)，使用虚拟环境可以让我们的包管理更加的方便，也为以后项目上线需要安装哪些包做好了准备工作。

  3. 虚拟环境相关操作：

    • 创建虚拟环境：mkvirtualenv --python='[python3.6文件所在路径]' [虚拟环境名字]。比如mkvirtualenv --python='C:\Python36\python3.6' django_env。
    • 进入到虚拟环境：workon [虚拟环境名称]。比如workon django-env。
    • 退出虚拟环境：deactivate。

  4. 首先进入到虚拟环境workon django_env，然后通过pip install django==2.0安装django，以Django 2.0版本为例进行学习。

    PS：请确保你是在虚拟环境中进行安装， 否则会默认把包安装在你的系统级的python环境中。

    判断进入虚拟环境的方法：命令行执行命令时，盘符前面是否有虚拟环境的名称。

  5. 安装pycharm profession 版本或者Sublime Text 3等任意一款你喜欢的编辑器。（推荐使用pycharm）。如果使用pycharm，切记一定要下载profession（专业版），community（社区版）不能用于网页开发。

  6. 安装最新版MySQL，windows版的MySQL的下载地址是：https://dev.mysql.com/downloads/windows/installer/5.7.html。如果你用的是其他操作系统，那么可以来到这个界面选择具体的MySQL来进行下载：https://dev.mysql.com/downloads/mysql/。

  7. 安装pymysql，这个库是Python来操作数据库的。安装方式也比较简单，pip install pymysql就可以啦。

如果你做完了上面7步的话，那我不得不说：恭喜你，你已经搞定了最让人头疼的环境问题！在下一章节里，我会告诉你，什么是Django。
